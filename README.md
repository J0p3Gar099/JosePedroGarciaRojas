# Portfolio — Jose Pedro Garcia Rojas

## Stack
- **Astro 4** — framework estático ultra rápido
- **CSS puro** — sin dependencias, animaciones CSS + JS vanilla
- **Google Fonts** — Instrument Serif + Geist

## Instalación

```bash
npm install
npm run dev
```

Abre http://localhost:4321

## Build para producción

```bash
npm run build
npm run preview
```

## Deploy (gratis)
- **Vercel**: `vercel deploy`
- **Netlify**: arrastra la carpeta `dist/`
- **GitHub Pages**: habilita en Settings → Pages

## Personalización
- Edita `src/pages/index.astro` para cambiar contenido
- Colores en `src/layouts/Layout.astro` → `:root { ... }`
- Agrega secciones copiando los patrones existentes
